## Grid Rotation

Post in **M → C → A → M → C → A** order. Every 3rd post cycles back to the same pillar, so the grid reads like a magazine.

Once you've posted 9 times, the grid shows a clean diagonal pattern (see mockup).

---

## Cadence

- **Minimum:** 3 posts/week (one per pillar)
- **Ideal:** 1 post/day, rotating through the 3 pillars
- **Reels:** always fall into a pillar — no "random" reels
- **Stories:** anything goes (this is where BTS, polls, memes, reposts live)

**Best post times (Amman):** 8–10pm local. Friday 5–7pm is top slot.

## Captions — voice rules

- Short. One or two lines max.
- Mix English + Arabic freely — Ammani accent/slang when it lands
- Call fans **"الحارة"** (the neighborhood)
- Every post ends with handle: `@zalameh.69`
- Hashtags **in the first comment**, not the caption

**Standard hashtag block:**
```
#zalameh #ammanrap #arabrap #jordanmusic #عمان
```

Add per-pillar tags:
- Music: `#newmusic #hiphop #rap #singer`
- Cat: `#catsofinstagram #illustration #mascot`
- Amman: `#jabalamman #weibdeh #jordan #streetphotography`

## Rules you don't break

1. Every reel has the audio tag at t=0.
2. Every post belongs to exactly one pillar — no hybrids.
3. No filename-as-content (no "toota", "foxx", "zaza" etc. on screen).
4. Released tracks only when labeling songs (Khaliki / Zan Zan / Abd Almajeed).
5. Don't skip a pillar for 3+ days in a row — algorithm reads you as "music only."
